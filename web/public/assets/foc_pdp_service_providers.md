# main.foc_pdp_service_providers

Mainnet Filecoin service providers reconstructed from ServiceProviderRegistry and FWSS events. Includes registry identity, event-derived PDP offering state, parsed capability fields, and FWSS approval status.

- asset code: `https://github.com/davidgasquez/filecoin-data-portal/tree/main/next/assets/main/foc/pdp_service_providers.sql`
- rows: `27`

## Depends

- `raw.fevm_eth_logs_decoded`

## Tests

- `not_null(provider_id)`
- `unique(provider_id)`

## Columns

| column | type | description | tests |
|---|---|---|---|
| `provider_id` | `BIGINT` | Service provider registry identifier. | `not_null`, `unique` |
